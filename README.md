# QuickStock

QuickStock is a **client-side Fabric mod** designed to scan QuickShop signs and help players identify shops that need restocking.

## Features
- Scan nearby QuickShop sell shops
- Sort by lowest stock first
- Persistent restock list
- World highlight for selected shops
- Clean UI designed for economy servers

## Requirements
- Minecraft 1.21.8
- Fabric Loader
- Fabric API
- Mod Menu (optional, but recommended)

## Notes
- Client-side only
- Designed for servers using QuickShop
- Does not read container contents
